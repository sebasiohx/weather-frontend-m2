# App Clima de Chile (prototipo)

🔗[github.com/sebasiohx/weather-frontend-m2](https://github.com/sebasiohx/weather-frontend-m2)

## 📘 Características

- ✅ Muestra el clima actual de Satiago y las otras capitales de las regiones de Chile
- ✅ hecho con data falsa que simula la informacion del clima
- ✅ Desarrollado con Javascript para renderizar los datos en el DOM
- ✅ Hecho con Bootstrap para el diseño responsivo

## 🛠 Instalación

Sigue estos pasos para correr el proyecto localmente:

```bash
# 1. Clona el repositorio
git clone https://github.com/sebasiohx/weather-frontend-m2.git

# 2. Entra a la carpeta del proyecto
cd weather-frontend-m2

# 3. Abre el proyecto (sin dependencias, es vanilla JS)
# Opción A: Abre index.html directo en el navegador
# Opción B: Usa Live Server en VS Code
```

> ⚠️ **Requisitos previos:** Solo necesitas un navegador moderno. No se requiere Node.js ni npm.
